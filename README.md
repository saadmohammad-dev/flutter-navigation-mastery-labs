## 📌 Project Overview
This repository contains solutions for **Lab Assignment 2**, covering basic to advanced multi-screen navigation features in Flutter, including passing data, custom layouts, Tab bars, Navigation Drawers, form handling, and global state tracking.

## 🛠️ Tech Stack & Concepts Explored
- **Framework:** Flutter (Dart)
- **Navigation types:** Imperial Routes, Navigator 1.0/2.0 basics, `RouteSettings`
- **UI Architecture:** Tab Bars, Drawers, Grids, dynamic state handling
- **State Management:** Local StatefulWidgets & shared instances for global states
